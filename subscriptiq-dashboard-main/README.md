# SubscriptIQ: AI-Powered Subscription Dashboard

SubscriptIQ is a high-performance, enterprise-grade subscription management platform designed for modern SaaS businesses. It combines cutting-edge web technologies with AI-driven insights to provide a seamless billing experience for both administrators and customers.

## ✨ Premium Features

### 🤖 AI Billing Assistant
- **GPT-4 Integration**: A contextual AI chat interface that helps users understand their billing cycles, usage patterns, and invoice details.
- **Smart Suggestions**: Quick-action chips for common queries like "When is my next billing date?" or "Download last year's summary".
- **Glassmorphic UI**: A visually stunning chat interface with interactive typing animations and a smooth, responsive layout.

### 📊 Advanced Financial Control
- **Dynamic Plans Engine**: Interactive pricing tiers with "Most Popular" badges, 3D hover effects, and automatic discount calculations.
- **Regional Tax Compliance**: Built-in support for Global VAT, Indian GST/SGST, and US Sales Tax (Nexus-based logic).
- **Smart Invoicing**: Automated PDF generation with AI-powered transaction breakdowns and compliance watermarks.

### 🛡️ Enterprise-Grade Components
- **Standardized Search & Filters**: A unified search and filter architecture across all modules (Subscriptions, Payments, Invoices, Discounts, Products) for a cohesive user experience.
- **StatCard Visualization**: Professional metric cards with animated counters and trend indicators.
- **Glass-Card Data Tables**: High-density data tables featured in a modern glassmorphism aesthetic with subtle backdrop blurs.

## 🛠️ Technology Stack

- **Core**: React 18, Vite, TypeScript
- **Styling**: Vanilla CSS + Tailwind CSS (Core Design System)
- **Animations**: Framer Motion (Iterative 3D effects and micro-interactions)
- **Icons**: Lucide React
- **PDF Generation**: html2canvas + jsPDF
- **UI Components**: Radix UI + shadcn/ui primitives

## 🚀 Getting Started

Follow these steps to set up the dashboard locally:

```bash
# 1. Clone the repository
git clone <repository-url>

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`. Ensure your backend server is running concurrently for full API functionality.

## 🎨 Design Philosophy

SubscriptIQ is built on a "Premium-First" design philosophy:
- **Rich Aesthetics**: Vibrant tailored HSL colors, sleek dark modes, and deep glassmorphism.
- **Micro-Animations**: Every interaction—from a button hover to a page transition—is designed to feel fluid and alive.
- **Visual Scannability**: High-contrast typography and clear visual hierarchies ensure complex data remains easy to digest.
